# vue2-element-node-server

```
|—————— assets  存放字体、图片文件
        |—————— font 存放字体文件
        |—————— img  存放全局图片文件
|—————— components 组件存放目录
        |—————— common 公共组件
             |—————— home  主页面
             |—————— layout  侧边选择栏
             |—————— login  登录页面
        |—————— pages  各页面组件
|—————— router  路由
|—————— sass  存放全局样式文件
|—————— service  存放全局公共文件

```

## 技术栈

   node + koa + mongoose 

## 使用
### 安装
  `npm install`
### 运行
  `npm start`

## 安装包的介绍
chalk： 颜色插件，可用chalk.blue(‘hello world’)来改变颜色；
